    How to set them on your iPhone
    1. Open the worklish-wallpapers folder on your Mac, select the image files (⌘A), and AirDrop those — they save straight into Photos. Don't AirDrop the folder itself — it lands in Files, where it can't be set as wallpaper.
    2. Already stuck in Files/iCloud? On iPhone: Files → the folder → Select → Share → Save Images to copy them into Photos.
    3. Long-press your Lock Screen → tap ＋ → Photo Shuffle.
    4. Choose Select Photos Manually, pick the Worklish images, set Shuffle Frequency (On Lock / Hourly / Daily).
    5. Tap Add — your lock screen now rotates through your expressions.
